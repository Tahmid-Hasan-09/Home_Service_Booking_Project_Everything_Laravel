1.First Install Composer Via Command Line(Go To https://getcomposer.org/download/)
	npm install
	npm run dev
2.Download Xampp V3.2.4 Or Latest Package

3.Import home_service_booking.sql file from 'Home Service Booking' folder into your xampp.

4.Rename or copy your .env.example file to .env.
  Configure your .env file with your database name.

5.Run $php artisan serve in your command line.

6.Visit 127.0.0.1:8000

7.Enjoy the newly Developed Site!!!
